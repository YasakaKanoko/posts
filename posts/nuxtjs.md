::: details <samp>目录</samp>

[[TOC]]

Nuxt.js 是一个基于 Vue.js 的用于创建前端应用程序的 JavaScript 框架

## 开始

::: code-group

```sh[npm]
npm create nuxt my-nuxt-app
```

```sh[pnpm]
pnpm create nuxt my-nuxt-app
```

```sh[yarn]
yarn create nuxt my-nuxt-app
```

```sh[bun]
bun create nuxt my-nuxt-app
```

:::
